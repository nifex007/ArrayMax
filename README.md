# ArrayMax
##Algorithm ArrayMax in python, php and javascript.
###Pseudocode
```python
Algorithm arrayMax(A, n):
  Input: An array A storing n ≥ 1 integers.
  Output: The maximum element in A.
  currentMax ← A[0]
  for i ← 1 to n − 1 do
    if currentMax < A[i] then
      currentMax ← A[i]
  return currentMax 
  ```
